Project ini merupakan tugas praktikum Pemrograman Web 2 menggunakan Laravel untuk mengelola data buku perpustakaan.

* Dashboard Buku
* Search & Filter Buku
* Tambah Buku
* Edit Buku
* Hapus Buku
* Validation Rules Advanced
* Bulk Delete Buku
* Export Data Buku ke CSV
